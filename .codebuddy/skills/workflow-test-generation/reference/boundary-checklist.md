# 边界条件 Checklist

系统化的边界条件检查清单，确保测试覆盖全面。

## 1. 数值类型

| 边界 | 测试场景 |
|------|----------|
| **零值** | `0`, `0.0`, `-0.0` |
| **负数** | `-1`, `INT_MIN` |
| **最大值** | `INT_MAX`, `UINT_MAX`, `DBL_MAX` |
| **最小值** | `INT_MIN`, `DBL_MIN` |
| **溢出** | `MAX + 1`, `MIN - 1` |
| **精度** | 浮点比较误差、舍入 |

## 2. 字符串/文本

| 边界 | 测试场景 |
|------|----------|
| **空字符串** | `""` |
| **空白字符串** | `"   "`, `"\t\n"` |
| **单字符** | `"a"` |
| **超长字符串** | 超过预期长度限制 |
| **特殊字符** | Unicode、emoji、控制字符、`\0` |
| **注入字符** | SQL 注入、命令注入相关字符 |

## 3. 集合/容器

| 边界 | 测试场景 |
|------|----------|
| **空集合** | `[]`, `{}` |
| **单元素** | `[x]` |
| **大量元素** | 性能边界、内存边界 |
| **重复元素** | 去重逻辑是否正确 |
| **顺序敏感** | 有序 vs 无序断言 |
| **null 元素** | 集合中包含 null/nullptr |

## 4. 指针/引用/可空类型

| 边界 | 测试场景 |
|------|----------|
| **null/nullptr** | 空指针输入 |
| **悬空引用** | 已释放对象的引用 |
| **Optional 空值** | `std::nullopt`, `None`, `null` |

## 5. 时间/日期

| 边界 | 测试场景 |
|------|----------|
| **epoch** | `1970-01-01 00:00:00` |
| **闰年/闰秒** | `2024-02-29`, 闰秒处理 |
| **时区边界** | UTC vs 本地时间、夏令时 |
| **过去/未来** | 时间戳比较逻辑 |

## 6. 并发/异步

| 边界 | 测试场景 |
|------|----------|
| **竞态条件** | 多线程同时访问 |
| **死锁** | 锁顺序问题 |
| **超时** | 操作超时处理 |
| **重入** | 同一线程重复调用 |
| **幂等性** | 重复调用结果一致 |

## 7. 状态机/流程

| 边界 | 测试场景 |
|------|----------|
| **初始状态** | 未初始化时的行为 |
| **终态** | 已完成/已关闭后的操作 |
| **非法状态转换** | 跳过中间状态 |
| **重复操作** | 重复 init/close |

## 8. 错误处理

| 边界 | 测试场景 |
|------|----------|
| **非法输入** | 类型错误、格式错误 |
| **资源不足** | 内存/磁盘/连接数耗尽 |
| **依赖失败** | 下游服务不可用 |
| **异常传播** | 异常是否正确抛出/捕获 |

## 使用方法

根据被测代码的参数类型，选择适用的边界条件。

**示例**：函数接受 `int count` 和 `string name`：
- 数值边界：0, -1, INT_MAX
- 字符串边界：空字符串、超长字符串、特殊字符
